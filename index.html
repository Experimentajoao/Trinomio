Decomposição algébrica

(a + b)² = 25
a² = 9
2·a·b = 12
b² = 4
Soma: 9 + 12 + 4 = 25
Lado = (a + b) = 5

Observe: o quadrado grande de lado (a+b) contém exatamente um quadrado a², um quadrado b² e dois retângulos a·b.

Módulo 03

Acessibilidade & Feedback Multissensorial

Esta aplicação foi pensada para todos os estudantes. Conheça e ajuste os recursos de inclusão.

Vibração tátil

Use a Vibration API para sentir o feedback de acertos (vibração curta) e erros (três pulsos curtos). Funciona em smartphones compatíveis.

Intérprete de Libras

Vídeos com intérprete de Libras estarão disponíveis em janelas flutuantes para cada módulo. Conteúdo a ser inserido em fase futura.

Feedback sonoro

Beep agradável para acertos e tom descendente para erros, gerados via Web Audio API (sem arquivos externos).

Reduzir animações

Estudantes sensíveis a estímulos visuais podem desativar animações sem perder funcionalidades.
